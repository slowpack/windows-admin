## 特点

- 高效的交互方式，让使用人员可以跨模块的多线操作
- 提供主题配置文件，轻松实现个性化定制
- 精心设计的动效，让每一处的动画都干净利落
- 根据配置自动生成导航栏
- 支持全方位权限验证

## 分支说明

- `master` 框架源码分支，不含示例代码，可直接用于实际开发
- `example` 演示源码分支，同演示站，包含大量示例，可用于参考学习
